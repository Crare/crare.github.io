3D bomber game inspired by <a href="https://fi.wikipedia.org/wiki/Bomberman">Bomberman</a> and <a href="https://fi.wikipedia.org/wiki/Minecraft">Minecraft</a>.
It was created on my free time when I was studying at Haaga-Helia University of Applied Sciences for a Bachelor's degree in Business Administration with speciliazation in software development back in 2015-2018. I scheduled few hours a day on this project so it did get quite far in development. Though the game may seem simple, short and lacking in content, it has a lot of work put into it.

It's made with <a href="https://unity.com/">Unity</a>-game engine and using mostly C#-langugage and Unity libraries. 3D-assets I made using <a href="https://www.blender.org/">Blender</a> and sound effects where made using 8-bit sound maker <a href="https://sfxr.me/">sfxr</a>.

<a href="https://jukepoks1.itch.io/bomber">You can play the game here on itch.io.</a>

Mostly it was a good practice in C# and Unity and general programming.(In the university we learned Java at the time.) It's still a bare bones to be a complete game. It would need more work and design to be a fully fledged game. It's still fun though.

I learned a lot about how to make 3D-models autotile nicely as there was no autotiling addons at that time and I wanted to do it myself. World is generated with multiple methods used such as <a href="https://en.wikipedia.org/wiki/Perlin_noise">Perlin noise</a> and custom rules to make it more varied. It was also first game I used <a href="https://en.wikipedia.org/wiki/A*_search_algorithm">AStar-pathfinding</a>. Though the pathfinding never was used at this point yet, but it <a href="https://twitter.com/Jukepoks1/status/822161287882936321">was already working as you can see here</a>.
