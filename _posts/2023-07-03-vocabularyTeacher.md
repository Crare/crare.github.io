Project Date: ~2018

<a href="https://github.com/Crare/VocabularyTeacher">Source Code</a>

In 2018 while I was studying still at Haaga-Helia, I had a swedish lessons to do. I'm not the best in swedish, but it is one of the native languages here in Finland so it is teached at school. In University the topics are related to professional language used at work. I wanted to learn these easily and expand my programming to it too.

I created this website that you could add words in two languages and it would ask them from you and give results afterwards. It would also try to test the same words more often that you have failed to answer correctly. This is a useful tool for anyone trying to learn a new language and study for a word exam and pass it easily.

I could rework this to the github.io site too someday, because it really could be a static webpage.

This was a good learning experience for using web technologies such as Javascript, <a href="https://angularjs.org/">Angular.js</a> and <a href="https://nodejs.org/en">Node.js</a>. Also swedish!

### -- Here's some images from the web app:

First you add the words in two languages in same order, so it can map them right to each other. Each word goes to new line so they are separated correctly. You can set some settings on how many times it asks the same words till you pass it. And how many options it gives you to choose from so it is not so obvious to just choose from 2 choices for example. And some other settings as well.

![Picture 3](/assets/vocabularyTeacher.png)

When the testing starts, it will show either this where you have to write the word correctly in the field next to the word on one of the languages. There's hint to see the answer too. but you won't get a point and have to do it again later.

![Picture 3](/assets/vocabularyTeacher2.png)

Other way test is done is giving multiple answer choices to choose from.

![Picture 3](/assets/vocabularyTeacher3.png)

After testing is done, you can see the results of your test. This is useful when you have lots of words to learn. You can see easily what words you have to prep more.

![Picture 3](/assets/vocabularyTeacher4.png)
